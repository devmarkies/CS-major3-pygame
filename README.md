## Controls
Player 1:
- Move left - A
- Move right - D
- Jump - W
- Attack 1 - R
- Attack 2 - T

Player 2:
- Move left - Left arrow
- Move right - Right arrow
- Jump - Up arrow
- Attack 1 - Numpad 1
- Attack 2 - Numpad 2
